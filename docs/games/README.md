# Games

Documentation for games on the WRDLNKDN Games surface (Dashboard → Games).

## Game docs

- [Daily Word Puzzle](./daily-word-puzzle.md) — Daily 5-letter word puzzle
  (Wordle-style); one puzzle per day, 6 attempts, letter hints, optional share.

## Overview

Games are provided under the **Games** feature flag. Solo games (e.g. Daily
Word, Tetris, Blackjack) and multiplayer games (e.g. Chess, Scrabble) are
started from the Games page; sessions and results are stored in the game session
framework.
